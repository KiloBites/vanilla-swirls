Getting your Wires Crossed?

Wires are the arteries/veins of electronics, so pay attention!

  • The display can have 3-6 wires on it.
  • Only the one key wire determines which button to push.
  • Perform the correct action on this button to progress to the next stage. Complete all stages to disarm the module.
  • Pressing an incorrect button or performing the wrong action will reset the module back to stage 1.
  • Wires and button positions begin with the first at the top.

3 wires:

Stage 1:
If the last wire is blue, the key wire is the first wire.
Otherwise if there are no black wires, the key wire is the last wire.
Otherwise if every wire is a different colour, the key wire is the second wire.
Otherwise the key wire is the last wire.
Stage 2:
If there are exactly two red wires, the key wire is the other wire.
Otherwise if the last wire is white or black, the key wire is the first wire.
Otherwise if there is a yellow wire, the key wire is the last wire.
Otherwise the key wire is the second wire.
Stage 3:
If there is a yellow wire and no white wires, the key wire is the last wire.
Otherwise if the first wire is red, the key wire is the second wire.
Otherwise if the second wire is a different colour to the other two, the key wire is the first wire.
Otherwise the key wire is the last wire.
Stage 4:
If there are exactly two wires that are the same colour, the key wire is the other wire.
Otherwise if there is both a white wire and a non-white wire, the key wire is the last non-white wire.
Otherwise if there are no blue wires, the key wire is the second wire.
Otherwise the key wire is the first wire.
Stage 5:
If there are no white or black wire, the key wire is the first wire.
Otherwise if there is exactly one yellow wire, the key wire is the second wire.
Otherwise if there are at least two wires that are the same colour, the key wire is the last wire.
Otherwise the key wire has the colour of the first button with a matching wire colour.

4 wires:

Stage 1:
If the first and last wires are the same colour, the key wire is the third wire.
Otherwise if the second and last wires are the same colour, the key wire is the first wire.
Otherwise if the third and last wires are the same colour, the key wire is the second wire.
Otherwise the key wire is the third wire.
Stage 2:
If any pair of consecutive wires are the same colour, the key wire is the second wire.
Otherwise if exactly one wire other than the last is black, the key wire is the first wire.
Otherwise if exactly one wire other than the first is white, the key wire is immediately above that white wire.
Otherwise the key wire is the last wire.
Stage 3:
If there are no uniquely coloured wires, the key wire is the first wire.
Otherwise if there are no duplicate wire colours, the key wire is the last wire.
Otherwise if there are no red wires, the key wire is the first wire of a repeated colour.
Otherwise the key wire is teh second wire.
Stage 4:
If the first wire is black, the key wire is the last wire.
Otherwise if there are consecutive red and blue wires, the key wire is the second wire.
Otherwise if there is exactly one repeated wire colour, the key wire is the last wire of the repeated colour.
Otherwise the key wire is the first wire.
Stage 5:
If there are exactly two distinct wire colours, the key wire is the first that is not the colour of the first wire.
Otherwise if the both the second and third wires are uniquely coloured, the key wire is the last wire.
Otherwise if there are no white wires, the key wire is the third wire.
Otherwise the key wire has the colour of the last button with a matching wire colour.

5 wires:

Stage 1:
If there is exactly one black wire and more than one blue wire, the key wire is the third wire.
Otherwise if there are no blue or yellow wires, the key wire is the last wire.
Otherwise if the first wire is uniquely coloured, the key wire is the last wire.
Otherwise the key wire is the fourth wire.
Stage 2:
If there is exactly one uniquely coloured wire, the key wire is that wire.
Otherwise if there are exactly three wires of the same colour, the key wire is the first wire not of that colour.
Otherwise if any of the middle three wires is white, the key wire is the second wire.
Otherwise the key wire is the first wire.
Stage 3:
If there are no uniquely coloured wires, the key wire is the last wire.
Otherwise if there is an equal number of red and white wires, the key wire has the same position as the button labelled 5.
Otherwise if there are no yellow wires, the key wire is the fourth wire.
Otherwise the key wire is the Rth wire, where R is the label of the red button.
Stage 4:
If there are exactly four distinct wire colours, the key wire is the Dth wire, where D is the label of the button whose colour is the repeated wire colour.
Otherwise if the first and last wires are the same colour, or exactly one of the middle three wires is red, the key wire is the third wire.
Otherwise if there are no wires the colour of the button labelled 1, the key wire has the same position as the button labelled 2.
Otherwise the key wire has the same position as the blue button.
Stage 5:
If the third wire, and no other wires, is blue, the key wire is the first wire.
Otherwise if the are no white wires, the key wire is the Bth wire, where B is the label of the blue button.
Otherwise if no wire matches its colour with the button in its position, the key wire is in the position of the button labelled 4.
Otherwise the key wire is the Mth wire, where M is the label of the first button to match its colour with the wire in its position.

6 wires:

Stage 1:
If more than one wire colour occurs exactly twice, the key wire is the third wire.
Otherwise if the last wire and at least one other wire is yellow, the key wire is the second wire.
Otherwise if there are no black wires, the key wire is the last wire.
Otherwise the key wire is the fifth wire.
Stage 2:
If exactly three wires are red or black, the key wire is the last wire of a repeated colour.
Otherwise if any wire colour occurs at least three times, the key wire is the first wire.
Otherwise if exactly one wire is yellow, and it is not the last wire, the key wire is immediately after the yellow wire.
Otherwise the key wire is the fourth wire.
Stage 3:
If there are any consecutive blue and yellow wires, the key wire is the fifth wire.
Otherwise if there at least two red wires, none of which are consecutive, the key wire is the first wire.
Otherwise if exactly one wire colour occurs exactly twice, the key wire is the first of those wires.
Otherwise the key wire is the Lth wire, where L is the label of the button that matches the colour of the last wire.
Stage 4:
If there is exactly one uniquely coloured wire, the key wire is the second wire.
Otherwise if every three consecutive wires has a white or black wire, the key wire is the last wire of a repeated colour.
Otherwise if no wire has the last button's colour, the key wire is in the position of the black button.
Otherwise the key wire is the third wire.
Stage 5:
If there are exactly four distinct wire colours, the key wire is the Xth wire, wherw X is the label of the button whose colour matches no wires.
Otherwise if the last wire is uniquely coloured, the key wire is the Lth wire, where L is the label of the button that matches the colour of the last wire.
Otherwise if there are no uniquely coloured wires, the key wire is in the position of the button with the lowest label whose colour matches a wire.
Otherwise the key wire is in the position of the button with the highest label whose colour matches a wire.

Button to Press:

Stage 1:
If the key wire is connected to A, press the button in the second position.
If the key wire is connected to B, press the red button.
If the key wire is connected to C, press the button labelled 4.
If the key wire is connected to D, press the button in the fourth position.
If the key wire is connected to E, press the yellow button.
If the key wire is connected to F, press the button labelled 1.
Stage 2:
If the key wire is connected to A, press the button with the same colour as the key wire in stage 1.
If the key wire is connected to B, press the button with the same colour that was pressed in stage 1.
If the key wire is connected to C, press the button on the fifth position.
If the key wire is connected to D, press the button with the same label that was pressed in stage 1.
If the key wire is connected to E, press the button labelled 2.
If the key wire is connected to F, press the black button.
Stage 3:
If the key wire is connected to A, press the button labelled 5.
If the key wire is connected to B, press the button in the same position that was pressed in stage 1.
If the key wire is connected to C, press the button with the same colour as the key wire in stage 2.
If the key wire is connected to D: if the the button in stage 1 was immediately released, press the black button, otherwise press the button with the same colour that light flashed in stage 1.
If the key wire is connected to E: if the the button in stage 2 was immediately released, press the black button, otherwise press the button with the same colour that light flashed in stage 2.
If the key wire is connected to F, press the button with the same label that was pressed in stage 2.
Stage 4:
If the key wire is connected to A, press the white button.
If the key wire is connected to B, press the button with the same colour that was pressed in stage 2.
If the key wire is connected to C, press the button with the same label that was pressed in stage 3.
If the key wire is connected to D, press the button in the same position that was pressed in stage 2.
If the key wire is connected to E, press the button labelled 3.
If the key wire is connected to F: if the the button in stage 3 was immediately released, press the black button, otherwise press the button with the same colour that light flashed in stage 3.
Stage 5:
If the key wire is connected to A, press the button with the same label that was pressed in stage 4.
If the key wire is connected to B, press the button with the same colour that was pressed in stage 3.
If the key wire is connected to C: if the the button in stage 4 was immediately released, press the black button, otherwise press the button with the same colour that light flashed in stage 4.
If the key wire is connected to D, press the button with the same colour as the key wire in stage 4.
If the key wire is connected to E, press the button with the same colour as the key wire in stage 3.
If the key wire is connected to F, press the button in the same position that was pressed in stage 3.

Action:

Stage 1:
If three or more wires match the colour of the button, press and immediately release the button.
Otherwise if the key wire matches the colour of the button, hold the button.
Otherwise if the button is labelled with its position, press and immediately release the button.
Otherwise hold the button.
Stage 2:
If there are no wires that match the colour of the button or the colour of the button that was pressed in stage 1, press and immediately release the button.
Otherwise if the button in stage 1 was immediately released, hold the button. Otherwise if the button is labelled with the position of the key wire, press and immediately release the button. Otherwise hold the button.
Stage 3:
If no wires match the colour of the button, nor the buttons pressed in stage 1 and 2, press and immediately release the button.
Otherwise if the button and the buttons pressed in stages 1 and 2 are uniquely coloured or labelled, hold the button.
Otherwise if the key wire and the key wires in stage 1 and 2 are uniquely coloured, press and immediately release the button.
Otherwise hold the button.
Stage 4:
If the button has both a position and label that have not been pressed yet, press and immediately release the button.
Otherwise if any light in a previous stage has flashed the colour of the button, hold the button. Otherwise if the button is not black nor has the black button been pressed in stages 1, 2, or 3, press and immediately release the button.
Otherwise hold the button.
Stage 5:
Hold the button.

Releasing a Held Button:

Stage 1:
If the light flashes red, release the button when the timer has a 1 in any position.
If the light flashes blue, release the button when the timer has a 4 in any position.
If the light flashes yellow, release the button when the timer has a 5 in any position.
If the light flashes white, release the button when the timer has a 3 in any position.
Stage 2:
If the light flashes red, release the button when the timer has the label of the button that was pressed in stage 1 in any position.
If the light flashes blue, release the button when the timer has the position of the button labelled 2 in any position.
If the light flashes yellow, release the button when the timer has the label of the last button in any position.
If the light flashes white, release the button when the timer has the position of the black button in any position.
Stage 3:
If the light flashes red, release the button when the timer has the position of the button that is the same colour as the key wire in any position.
If the light flashes blue, release the button when the timer has the position of the button that has the same colour as the button that was pressed in stage 2 in any position.
If the light flashes yellow, release the button when the timer has the position of the button that has the same label as the button that was pressed in stage 1 in any position.
If the light flashes white, release the button when the timer has the label of the button in the same position as the button that was pressed in stage 2 in any position.
Stage 4:
If the light flashes red, release the button when the timer has the position of the button that has the same label as the button that was pressed in stage 3 in any position.
If the light flashes blue, release the button when the timer has the position of the button that has the same colour as the key wire in stage 2 in any position.
If the light flashes yellow, release the button when the timer has the label of the button that has the same colour as the button that was pressed in stage 3 in any position.
If the light flashes white, release the button when the timer has the label of the button that has the same colour as the button that was pressed in stage 1 in any position.
Stage 5:
If the light flashes red, release the button when the timer has the label of the button that has the same colour as the key wire in stage 4 in any position.
If the light flashes blue, release the button when the timer has the label of the button that has the same colour as the key wire in stage 2 in any position.
If the light flashes yellow, release the button when the timer has the label of the button that has the same colour as the key wire in stage 1 in any position.
If the light flashes white, release the button when the timer has the label of the button that has the same colour as the key wire in stage 3 in any position.
